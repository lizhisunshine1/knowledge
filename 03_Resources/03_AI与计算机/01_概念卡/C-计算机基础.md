# C-计算机基础

## 定义
计算机基础在此指代为了更好地开发自动化工作流、编写代码或与外部 AI 接口交互所必备的核心底层概念，如 API（应用程序接口）、JSON（数据流格式）、HTTP 请求、环境变量和路径规范等。

## 通俗解释
不懂计算机基础，就像开赛车却不懂发动机和换挡原理。要想真正让 AI 为你自动干活，你必须理解软件与软件之间是如何通过 API “打电话”通信的，如何用 JSON 格式给对方发信，以及电脑如何通过环境变量记住敏感的安全密钥。

## 关键区别
* **纯小白视角**：只通过网页端的对话框使用 AI，无法进行系统集成和批量化提效。
* **计算机基础赋能**：掌握底层机制，能自由拼装大模型、脚本与各类工具，实现真正的“降维打击”。
