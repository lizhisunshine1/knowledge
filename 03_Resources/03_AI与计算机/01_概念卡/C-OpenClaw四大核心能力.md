# C-OpenClaw四大核心能力

## 这个概念的准确定义是什么？

OpenClaw（龙虾）相比传统AI助手的4个本质不同：

1. **操控电脑** — 有"电脑手"，能操作浏览器、读写文件、发邮件
2. **接入聊天软件** — 人在外手机发消息，远程指挥电脑干活
3. **长期记忆** — 记忆存为Markdown文件，偏好惯例永远记住
4. **Skills** — 常做任务固定为流程文档，说关键词即执行

## 它像什么？

一个能干的实习生——你交代它去做，它会越做越顺手。但实习生不能帮你"找到更值得做的事"。
