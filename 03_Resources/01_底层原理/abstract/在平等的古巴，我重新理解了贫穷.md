# "在平等的古巴，我重新理解了贫穷" 摘要

## 核心论点

"大家都一样穷"并非理想社会。古巴作为绝对平均主义的现实样板，证明了一条铁律：**绝对的均，长期来看会导致绝对的寡。**

---

## 框架/模型

### 1. 三次分配框架（刘润版）

| 分配次序 | 目标 | 主导者 | 关键词 |
|---------|------|--------|--------|
| 第一次 | 把蛋糕做大（效率） | 商业/市场 | 收入和价格指挥棒 |
| 第二次 | 分好蛋糕（公平） | 政府 | 税收、补贴、救济 |
| 第三次 | 查漏补缺 | 公益/慈善 | 润滑社会运转 |

**核心观点：顺序不能乱。** 古巴的问题在于第一次分配追求的不是效率而是公平。

### 2. "自我削弱的调节回路"（系统思维）

```
没人种甘蔗 → 没有糖 → 没有外汇
  ↑                        ↓
没有燃油 ← 买不到农机 ← 无外汇进口
  ↓
榨糖厂开不了机 → 更没有糖 → (循环加强)
```

### 3. 激励机制铁律
**当多劳不再指向多得，愿意多劳的人就会消失。**

## 关键案例

| 案例 | 要点 |
|------|------|
| 农贸市场 vs 供销社 | 市场价高但有货 vs 补贴价低但空架 |
| 维修一次性打火机 | 通缩环境下，一次性商品变成了传代资产 |
| 古巴蔗糖业 | 800万吨/年 → 不到20万吨/年（1/40） |
| 鸡蛋成为"战略物资" | 严禁私卖，导致供销社和黑市都没有 |
