## 核心论点

古巴的经济困境不全是美国封锁造成的，内部"去工业化"的连锁反应才是根源。

---

## 框架/模型

### 1. 去工业化连锁反应
```
没人种蔗 → 无糖 → 无外汇 → 无设备/燃油
  ↑                            ↓
  ← ← ← 产能萎缩 ← ← ← 工厂停工
```

### 2. 资源诅咒
- 日产3万桶石油，但全是劣质超重油
- 缺少炼油技术 → 只能烧油发电 → 腐蚀设备 → 天天停电

### 3. 汇率三重扭曲
- 官方：1:24（企业）/ 1:120（民众）/ 1:20（特殊）
- 黑市：1:500
- 政府通过汇率差榨取外汇价值：1000美元出口 → 工人手中仅值48美元
