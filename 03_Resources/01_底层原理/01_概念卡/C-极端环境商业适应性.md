## 定义
在货币失效、资源匮乏、基础设施不可靠的极端环境中，市场自发产生的替代性商业模式。这些模式往往绕开管制、利用信息差、创造新的交易媒介。

## 核心模式清单
| 模式 | 本质 |
|------|------|
| 职业排队人 | 时间证券化——排队权交易 |
| 每周包裹 | 离线互联网——分发网络替代宽带 |
| 话费=外汇 | 金融创新——话费余额成为准货币 |
| 阳台微商 | 极小微零售——住宅延伸为店铺 |
| 人肉骡子 | 碎片化供应链——行李箱式进口 |
| 离线电影院 | 停电经济——备用电源+投影仪 |
| WiFi二房东 | 套利——批发出售总量 |
| 社区辛迪加 | 去中心平台——个人=Airbnb+携程 |
| WhatsApp易货 | 物物交换——回归以使用价值计价 |
