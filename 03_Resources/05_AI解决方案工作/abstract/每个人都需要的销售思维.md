## 核心论点

销售是一套可训练的话术流程，掌握五步法即可覆盖80%的谈单场景。

---

## 框架/模型

### 1. 销售五步法

```
开场白 → 挖需求 → 介绍产品 → 解决异议 → 促单成交
```

### 2. 开场白"晒价值"四步
1. 确认信息（你是我要找的人吗）
2. 展示实力（我是谁，我多牛）
3. 相似案例（隔壁李先生的装修效果）
4. 号召行动（加个微信？见一面？）

### 3. 挖需求：问四类背景问题
- **问钱**：预算多少？
- **问权**：谁当家作主？
- **问需求**：不满在哪？想改变什么？
- **问竞品**：还看了哪家？过去用过啥？

### 4. 价格异议三招
- **价格锚点**：推出更贵的版本让中间档显得合理
- **减法**：折算附加价值后的实际价格
- **除法**：总价÷天数→每天只要X元

### 5. 促单三原则
① 主动（销冠没有"意向客户"，要么签要么弃）
② 客户利益（建立长期信任，不是一次性买卖）
③ 理性+感性（逻辑说清，情绪收尾）
