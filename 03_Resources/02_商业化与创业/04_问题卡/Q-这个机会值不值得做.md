# Q-这个机会值不值得做

## 问题定义
面对一个新发现的商机（比如：为某传统行业定制一个大模型客服系统，或者针对小红书做AI配图工具），如何迅速判断是否值得投入个人的核心精力去推进。

## 关键判断标准

### 三大核心问题

1. **做成后的回报，是否明显大于代价？**
   - 回报不够大 → 再能做成也不值得投入

2. **失败了，损失是否可控？**
   - 伤到主业/现金流/信誉/团队 → 谨慎
   - 只损失几周时间和几千块 → 值得试

3. **现在做，是否比以后做更有优势？**
   - 窗口期打开 → 早点进入
   - 还太早 → 小验证，不重投入

### 机会价值公式
```
价值 = 市场空间 × 付费意愿 × 胜率 × 可复制性 - 下行风险
```

## 常见方案
* ❌ 方案A：凭直觉和一时冲动，立刻辞职或者砸钱开发系统（极其危险，九死一生）。
* ✔️ 方案B：**使用五维评估法与MVP验证流程**。先算清期望值，用最轻量的动作（如：售前方案PPT或朋友圈意向测试）去探知市场的真实反应。
