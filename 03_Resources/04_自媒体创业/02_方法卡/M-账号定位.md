# M-账号定位

## 适用问题
如何为自己的自媒体号找到兼具**个人优势**、**用户痛点需求**和**可持续变现闭环**的“黄金生态位”，避免因为定位模糊导致起号失败或“空有粉丝无法变现”。

## 操作步骤
找到以下三个圆圈的交集（即 Sweet Spot）：

```text
       ┌───────────────┐
       │   个人优势与  │
       │   专业认知高地│
       └───────┬───────┘
               │
       ┌───────┴───────┐
 ┌─────┴─────┐   ┌─────┴─────┐
 │ 目标用户群│   │ 明确可行的│
 │ 的痛点需求│   │ 商业变现路│
 └───────────┘   └───────────┘
```

1. **圈A：个人优势**：
   - 我在什么事情上比 80% 的普通人做得更好、更有经验？（如：高级AI解决方案架构、PPT制作、工作流配置）。
2. **圈B：用户需求**：
   - 市场上有什么痛点是人们每天都在抱怨且渴望解决的？（如：打工人想下班提效、公司想降低AI落地成本）。
3. **圈C：变现路径**：
   - 什么样的内容产出能直接指向变现产品？（如：售卖咨询服务、大模型知识库方案PPT模板、付费社群）。

## 注意事项
* **定位要垂直**：千万不要今天写AI，明天写旅游，后天写美食。这会让算法无法识别你的标签，读者也无法建立深度信任。
* **离钱近一点**：在做定位的第一天，就必须想清楚“我最终靠什么收钱”。
