# 周复盘模板

> 每周五花 30~45 分钟执行。

## 第 1 步：清 Inbox
把所有未处理输入逐条决定去向：
- 项目 → `01_Projects`
- 领域 → `02_Areas`
- 资源 → `03_Resources`
- 归档 → `05_Archives`

## 第 2 步：看项目
每个活跃项目只看三件事：
- 进展
- 阻碍
- 下一步

## 第 3 步：看输出
本周有没有形成可复用成果？
如果没有，至少要把一条知识整理成卡片。

## 第 4 步：做渐进式披露
挑 1 到 3 条高价值笔记继续压缩：
- 原文 → 高亮 → 自己的话 → 可调用版本

## 第 5 步：定下周重点
只定 3 件最重要的事。
