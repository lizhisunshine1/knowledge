# ⚡ 触发器思维

> 当遇到某类问题时，自动触发对应卡片组。
> 训练大脑形成条件反射。

---

## 触发器 1：遇到商业机会

```
→ 打开 [[99_System/09_问题索引页]] 找 Q1
→ 调取卡组：
    原则：P-先验证再投入 · P-高痛点优先 · P-小步快跑 · P-长期主义
    概念：C-需求 · C-商机判断 · C-商业模式
    方法：M-如何发现真实需求 · M-如何做最小可行验证
    框架：[[06_Opportunity_Lab/机会判断方法论·完整框架]]
→ 输出：一张 O-机会卡 → 评分 → 决定验证
```
