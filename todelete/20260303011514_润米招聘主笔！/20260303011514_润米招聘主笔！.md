01

**我们在找什么样的人？**

经常有候选人问我：“我的文章文采斐然，逻辑通顺，为什么还是被毙了？”

通常我会问一句：“读完你的文章，读者是'哦，知道了'，还是'哇，原来如此'？”

前者，是信息的搬运工；后者，才是我们要找的——认知代理人。

为什么这么说？

我们的读者是中国的创业者和企业管理者。他们的时间比金钱更贵。如果你浪费了他们5分钟，却只给了一个网上能搜到的结论，那是在“谋财害命”。

我们的文章，是能帮他们节省认知成本的。

**我们的主笔，生产的不是文章，是“认知交付”。**

什么是“认知交付”？我们用一道“菜”来理解。

写文章，就像是做一道大菜。

一道真正的大菜，必须包含四个部分：主菜（肉）、配菜（菜）、调味（盐）、和厨艺（手艺）。

**好文章 = 认知差(肉) + 信息差(菜) + 情绪(盐) × 易读性(手艺)**

![Image](images/img_002.png)

没有肉，这道菜不成立。有肉无菜，会腻。有菜无肉，会饿。再好的食材，没有盐，也寡淡无味。而手艺，决定了这道菜是焦黑一团，还是色香味俱全。

我们要找的，是能做“认知大餐”的厨师。

02

**公众号主笔岗位具体要做什么？**

岗位职责：

1. **做“主菜”，提炼认知差**

- 基于商业素材，不只是整理信息，而是提炼出让读者“原来如此”的洞察
- 把零散现象，变成可复用的认知模型

2. **配“好菜”，挖掘信息差**

- 采访企业家、投资人，拿到网上搜不到的一手素材
- 解读商业报告，找到别人没注意到的关键数据

3. **调“味道”，拿捏情绪**

- 用共鸣、反差、悬念等，让读者愿意读完、记住、传播
- 帮读者说出他心里想但说不出的那句话

4. **秀“手艺”，保证易读性**

- 逻辑严谨，观点聚焦，结构清晰，金句点睛
- 和我们已有的文章风格保持一致

岗位要求：

1. **有表达欲，但更理性，追求深度**

- 文字能力：逻辑严谨，观点聚焦，拿捏用词。
- 熟悉商业和企业管理的基本知识体系，真心喜欢商业话题（我们日更，不喜欢会很难坚持）。

2. **对互联网商业热点敏感**

- 能及时把握创作题材，能问出好问题、找到好角度。

3. **有主人翁意识**

- 你写的不是“公司的文章”，是“你的认知交付作品”。
- 事事有反馈，件件有着落，眼里有活，心中有作品。

4. **有作品**

- 投递简历时，务必附上商业或企业管理方向的已有文章作品。
- 如有试稿，请记录具体创作流程（先后顺序及大概用时）。

1. 足够的空间：我们不告诉你“必须这么写”，而是问：“你觉得怎么写最好？”
2. 真实的战场：你每天接触的是一线创业者、投资人，不是“媒体通稿”式的采访，是真正有深度的对话。
3. 成长的速度：在这里，一年可能抵得上别的地方三年。因为你要独立负责的事，比别的地方多得多。
4. 超预期的回报：

- 早九晚六，周末双休，五险一金实缴。
- 固定薪酬 + 爆文奖 + 年底绩效奖，优异者奖金非常丰厚。
