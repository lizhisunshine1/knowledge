# 如何建立一个高质量的人脉圈？

#### 有这么一句话，我一直觉得挺有道理。

**身边是什么样的人，我们就会变成什么样的人。**

他们逃课，我们可能就会跟着一起逃课。他们垂头丧气，我们也会感觉四季如冬。他们充满能量，我们才会满怀信心。

这也是为什么，很多家长挤破头，要把孩子送进一所好学校，很多同学拼了命，要考进一所好大学，很多伙伴费大力，要融入一些所谓的圈子。

因为，无论是发展个人，还是成就事业，或多或少都离不开身边人的帮助。关键时刻，能有贵人的指点、提携，也真的是件很幸运的事。

**但是，如何才能建立一个高质量的人脉圈？怎么才能找到我们自己的贵人？**

前段时间，我们的读书会栏目，邀请到了孙路弘老师。

孙路弘老师，是著名的营销及销售行为专家，科特勒营销集团中国区的高级顾问，常被称为“中国销售培训第一人”。他写的《用脑拿订单》，也是很多销售人的枕边书。可以说，多年以来，孙老师的核心工作之一，就是研究“如何与人打交道”。因为销售，本质上就是一门“和人打交道”的学问。

同时，数学和逻辑的背景，又为孙老师提供了更多的分析维度。

而这次，围绕着新书《连接就是竞争力》，孙路弘老师分享了很多。关于沟通。关于建立人脉圈。关于寻找贵人。一场直播下来，收获颇丰。

所以今天，请允许我，与你分享。

![Image](images/img_002.png)![图片](images/img_003.png)**很多所谓的人脉，其实只停在“交换信息”的层面**

首先，什么是人脉？

你和他在同一个群里，互相加了微信，发了名片，那他是你的人脉吗？

不是的。

**孙路弘老师说，人际关系，其实也有“任督二脉”。**
^s1-1


**督脉，是阳脉，也就是一个人的公开信息。**

比如，他的名片上写着，张三，A公司总经理，电话号码，邮箱地址。这些信息，是硬邦邦的，摆在台面上的。“张总，最近生意怎么样”，“李工，那个项目进展如何”。这些话题，也都是“督脉”层面的话题。你们交换的，是信息，是数据，是逻辑。这样的沟通，更像是一种“交谈”。

我们这一生，会遇到很多的人。和多数人的关系，其实会一直停留在“督脉”这个层面。这很正常。

但是，只有督脉的关系，脆弱得像一张纸。业务纽带断了，离职了，退休了，这张纸也就破了。

**任脉，是阴脉，也就是一个人的隐私信息、情感流动。**

比如，你知道张总虽然在工作上得心应手，但在孩子的教育问题上常常愁得睡不着觉。你知道李工虽然是技术专家，但其实一直有一个摇滚梦。聊起这些，你们就不是“交谈”，而是“交流”了。

人和人的连接，不能只靠信息的搬运，也要靠情感的交换。

**而真正的人脉，是任督二脉都要通。**

很多人的微信列表里，有几千个头像。但遇到挫折、需要帮助、寻求指点的时候，却常常不知道该找谁。本质上，就是只拥有了一个巨大的“信息库”，而不拥有一个温暖的“能量场”。

所以，建立人脉，不仅要发名片，也要去感知对方的任脉。不仅要交谈公事，也要去交流情感。

因为人脉，永远生长在情绪的土壤里。

### 图片

**人际关系，也是可以被“体检”的**
^s1-2


可是，我怎么知道，我和某个人的关系，到底通没通？通到了什么程度？

的确。很多时候，我们都是在用“感觉”回答这些问题。我觉得我和老王的关系挺好的，我们还经常一起吃饭来着。我觉得我和我爱人没什么问题，都相处了这么多年了。

但是，感觉，未必是准确的。

**一个医生，不能只听一个人说“我觉得自己很健康”。他会让这个人去体检，让数据说话。**

可是，人际关系，要怎么被“体检”？

举个例子。

有一次，一位读者，买了孙路弘老师的书。她觉得，自己和爱人的关系特别好。如果打分的话，能打98分。

但是，面对书中一个量表里的问题时，她懵了。这个量表里，都是一些并不复杂，但更加深层的问题。比如，他上学的时候，最讨厌的老师是谁。比如，他小的时候，最害怕什么东西。比如，他现在的工作里，最让他感到无力的事情是什么。

这个时候，她才发现，她知道爱人爱吃红烧肉，知道爱人的工资，却很少知道爱人的精神世界。

如果按照量表的标准打分，那这段关系，其实是不及格的。

是的。

**我们很容易把“时间长”，当成“关系深”。把“物理距离近”，当成“心理距离近”。**

所以，父母、伴侣、合作伙伴、核心客户，这些你在乎的关系，其实都值得你“体检”一遍。

他最近在烦什么？他会为一个什么样的瞬间而骄傲？他看重什么样的品质？这些问题，如果你只能回答“我觉得”，而不是“他曾经告诉我”。那么，这段关系，也许未必有你想象得那么好。

**了解得越浅，越像“观众”。了解得越深，才越是“自己人”。**

### 图片

**了解一个人，可以从半公开信息开始**
^s1-3


体检完了。关系不深。怎么办？

我总不能直接说，“请你告诉我你的童年阴影”吧？

**当然。沟通，也是有节奏的。孙路弘老师，就提到了一个“洋葱模型”。**

一个人的最外层，是皮，是公开信息。这层皮，谁都能看，没什么营养。最内层，是芯，是隐私信息。这层芯，容易辣眼睛，不能随便剥。但是，中间还有一层，是肉，是半公开信息。

什么是半公开信息？

比如你的爱好，比如你的习惯，比如你最近读的一本书。这些信息，不涉及核心隐私，但又展示着你的个性。

举个例子。

当年，孙路弘老师年轻的时候，在国外留学，去肯德基打工，人家让他去做“鸡手”。他很纳闷，什么是鸡手？为什么要做鸡手？炸鸡翅的人，在这里都叫鸡手吗？

后来才知道，闹笑话了。因为当时英语不好，他把Kitchen Hand，听成了Chicken Hand。

这个故事，就是一个半公开信息。了解一个人，就可以从这样的信息开始。

**同样的，当你开始分享自己的半公开信息的时候，对方也会放下戒备。**

所以，我们也可以梳理出一些自己的半公开信息，作为沟通的素材。比如，我其实是一个路痴，在商场里也会迷路。比如，我特别容易招蚊子。比如，我很喜欢下棋，虽然棋艺很臭。比如，我特别喜欢《三体》，看了三遍了。

这些信息，是一种强烈的信号。我愿意让你看到更加真实的我。

同时，分享这些信息，也能找到一种更加重要的东西。

### 图片

**同好，就是确认自己人的暗号**
^s1-4


什么更加重要的东西？

**孙路弘老师说，这个东西，就是“同好”。**

在传统的中国社会里，我们讲究“五同”。同乡，同宗，同姓，同学，同事。

过去，这“五同”非常管用。你是哪里人？山东的。哎呀，我也是山东的。老乡见老乡，两眼泪汪汪。关系，一下子就近了。但现在，“五同”开始不断失效。很多人集中在北上广深打拼，同乡并不少见。对很多大姓来说，同姓更是常见。而同事，又常常伴随着竞争关系。

这个时候，“同好”的价值，就越来越明显。

**因为“同好”，就是确认自己人的暗号。**

当我们发现，我们都喜欢同一个歌手，痴迷同一类电影，爱玩同一款游戏的时候，我们的大脑，就会分泌一种多巴胺。这种多巴胺会告诉我们，这个人是懂我的，是安全的，是我的同类。

所以，如果真要说，建立高质量人脉圈有什么捷径，那寻找、挖掘，甚至培养、创造“同好”，一定就是其中一条。

和一个重要的人见面的时候，试着打开你的感官。他的桌上，摆着什么样的茶具？书架上，是一些什么样的书？墙上，是一幅什么画？甚至，聊出一些门道来。

这样，你们的对话，就不会因为没有话题而尴尬。你们的交流，也会更加顺畅。

### 图片

**赞美，不一定是夸奖，也可以是请教**
^s1-5


分享半公开信息，找到同好。然后呢？怎么才能更进一步？

是不是要送礼，请客？

**不一定。请教，也是一种好办法。**

说到这里，孙路弘老师分享了一个销售员的故事。

有位销售员，想搞定一位财务总监。但是，这位总监油盐不进。

后来，他才知道，这位总监特别喜欢养鸟。于是，他去书店，买了一些关于养鸟的书。一周之后，他找到总监，真诚地问，王总，我最近想养只鸟，但遇上了一些难题，您能不能指点我一下？

总监一听，来劲了。这是我的专业领域呀。而且，平时聊的，都是财务报表，哪有人和他聊养鸟。于是，总监拉着销售员，聊了足足两个小时的养鸟经。

慢慢的，单子，也终于有了突破口。

是的。

**赞美，不一定是夸奖，也可以是请教。**

因为向一个人请教他擅长的领域时，你相当于也在说，1）我认可你的专业程度。你在这一块儿比我强，你是我的老师。2）我信任你的为人。我相信你会帮助我，而不是嘲笑我。3）我愿意在这个领域成为你的学生。

人，都是渴望被需要，渴望优越感的。

所以，准备一些真诚的小问题。比如，您家的灯真好看，在哪儿选的呀？比如，您的孩子真优秀，能不能也教我两招？

这些小小的请教，成本很低，但效果很好。

当然，有个前提。那就是，必须真诚，也必须具体。

搜索引擎就能回答的问题，是浪费时间的。“您是怎么成功的”这类问题，是回答不了的。

# 图片

# **每个人的委屈，都需要有人能懂**
^s1-6


那么，再然后呢？能不能再深一步？甚至，成为知己？

那就得有深度的情感共鸣了。

**用孙路弘老师的话来说就是，同情。**

当然。这种同情，不是“哎呀你真可怜”。“你真可怜”，是一种居高临下的状态。我比你强，我比你好，我施舍一些善意给你吧。这种关系，是不平等的。

这种同情，是一种共情，一种共振，一种感同身受。

因为很多时候，朋友向你倾诉，他并不期待你立刻给他三个解决方案，也不希望你说“哎呀这都小事儿”。他需要的，是你坐下来，告诉他“我懂。换了是我，我也会很难过”。

**看见下雨的时候，他需要一把雨伞。已经淋湿的时候，他希望自己并不孤单。**

每个人的委屈，都需要有人能懂。

这种情绪上的接纳和同步，是一种非常稀缺的资源。

因为在职场上，我们习惯了讲逻辑、讲效率、讲KPI。我们把自己包裹在厚厚的铠甲里。但是，如果你可以接纳别人的情绪，可以在别人焦虑的时候提供安抚，可以在别人悲伤的时候提供陪伴。那么，你就更容易打通“任脉”。

甚至，看清对方背后的“三维关系图”。

# Image

# **每个人的背后，都有一张巨大的网**
^s1-7


什么是“三维关系图”？

人，不是孤岛。每个人，其实都是某张巨大网络上的节点。他代表了这个点，也代表了身后的线，和线那头的网。

而这张网里，有两个重要的因素。

**影响力，和制约力。**

举个例子。

你想和一家公司合作，想要搞定他们的采购经理。因为你觉得，他是关键人物。

但是，你可能不知道的是，这位采购经理，虽然有决定权，但非常听他师傅的话。而他的师傅，又和你们公司的竞争对手处得很好。

如果只看见了点，你会觉得，采购经理已经很认可了，单子怎么还是飞了。但如果看见了网，你就会理解，真正决定是否合作的，到底是什么。

所以，“他”最在意的，是谁的看法？上司，爱人，还是老友？和谁是利益共同体？和谁是死对头？

**背后的网，往往比眼前的点更重要。**

读懂了关系网，你也就有了更加系统性的人脉思维。

# **做一个“吸引者”，而不是“狩猎者”**
^s1-8


好。学了这么多的方法。那么现在，是不是就能深交很多人，甚至找到自己的贵人了？

不是的。

**“找到贵人”的本质，其实是“自己成为了贵人”。**

什么意思？

就像WiFi热点。你是想满世界找WiFi连，还是想成为一个强信号的WiFi热点，让别人来找你？

人脉的本质，是价值交换。

这个价值，不一定全是金钱和利益。它可以是信息，因为你可以提供独家的行业洞察。它可以是情感，因为你可以提供极高的情绪价值。它可以是才华和潜力，因为你让人觉得未来可期。

**能认识多少人，很重要。多少人想认识你，更重要。**

所以，建立一个高质量的人脉圈，最根本的办法，其实是修炼内功。

你在你的专业领域，做到极致了吗？你有什么独特的资源，或者见解？

就像孙路弘老师说的那样。不要做一个“狩猎者”，而是要做一个“吸引者”。

狩猎者，拿着枪，追着兔子跑。吸引者，用追兔子的时间，种下一片草地。

春暖花开的时候，兔子们自然会来。

# Image

# **不要等口渴的时候，才想起来挖井**
^s1-9


当然。最后，还有最重要的一点，也是孙路弘老师反复强调的一点。

**关系，是存出来的，而不是透支出来的。**

很多人平时不烧香，一联系，就是麻烦。

这种急用的人脉，往往是最贵，最不可靠的。

因为人脉，是有时间复利属性的。开始的时间，最好是十年前。其次，就是现在。

不要等口渴的时候，才想起来挖井。没事的时候，也可以发个问候。不需要帮忙的时候，也可以主动提供价值。

**今天存一点关心，明天存一点帮助，后天存一点理解。**

这些存款，会在时间的酝酿下发酵，变成真正的情义。

风暴来了，也许不用你开口。曾经的存款，会主动成为你的大伞。

# **最后的话**

再次感谢孙路弘老师的分享。

现在，让我们回到最开始的问题。

**如何才能建立一个高质量的人脉圈？怎么才能找到我们自己的贵人？**

从任督二脉，到量表体检。从洋葱模型，到同好、请教、共情。从三维关系图，到WiFi热点。

同时，别忘记“真诚”和“时间”的力量。

当你愿意真诚地展示自己，当你懂得温暖别人，当你开始加强自己的信号，当你愿意长期耕耘，你就会发现，所谓的“高质量人脉圈”，自然就会生长出来。

至于贵人？或许也不必刻意寻找了。

因为你自己，就是那个值得被贵人连接，值得被贵人信任的，真正的贵人。

**你身处星空。而你自己，也是发光的星星。**

2026，希望我们，都能打通自己的任督二脉。

加油。
