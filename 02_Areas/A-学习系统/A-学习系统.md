# A-学习系统

## 领域定义
对个人“学习机器”与“认知获取流水线”本身的维护、调优与升级，即对“第二大脑”本身的持续关切。

## 理想状态
* 拥有极其流畅、无摩擦的“Capture -> Organize -> Distill -> Express”知识流转工作流。
* 能够将遇到的任何新鲜好内容在数天内消化、提炼并以卡片化形式沉淀进 Resources。
* 确保知识库处于“健康、低臃肿、高价值”状态。

## 持续性行动
* [ ] 每周五准时进行系统清理与复盘。
* [ ] 每月重构一次 MOC，剔除不再关注的垃圾资源。
* [ ] 定期盘点和优化自动化捕捉工具与插件的运行稳定性。
